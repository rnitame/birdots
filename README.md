# Install

```
$ git clone https://github.com/rnitame/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./install 
```

# Uninstall

```
$ cd ~/.dotfiles 
$ ./uninstall 
$ rm -rf ~/.dotfiles 
```
