# dotfiles

## Description
Install file list

- tmux
  - .tmux.conf
  - .tmux.conf.osx
- git
  - .gitconfig
  - .gitignore
  - .gitmessage
- zsh
  - .zshrc
- vim
  - .vimrc

## Requirement

- Vim 7.4 (+ lua)
- Zsh 5.2
- tmux 2.2
- git

## Install

```
$ git clone git@github.com:rnitame/birdots.git
$ cd birdots
$ ./land.sh
```
